What is Sisimai ?
=================

Sisimai is a core module of bounceHammer version. 4, is a Perl module for 
analyzing email bounce. "Sisimai" stands for SISI "Mail Analyzing Interface".

System requirements
-------------------

* Perl 5.10.1 or later

Dependencies
------------
Sisimai relies on:

* __Class::Accessor::Lite__
* __Try::Tiny__
* __JSON__
